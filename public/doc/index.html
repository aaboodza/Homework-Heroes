Installion


1. In order to set up and test a Node/Express locally on your system, you will first have to install Nodejs and the Node Package Manager (npm) on your operating system.

2. You can start the server by calling node with the script in your command prompt:

node index.js

3. Open Browser and type in http://localhost:3000 to run it locally or user our Live server

4. To stop the server, type ctrl + c in the terminal:
